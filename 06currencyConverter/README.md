# 💱 React Currency Converter

A simple and efficient Currency Converter Application built using React.
This project demonstrates real-time conversion using API data, state management, controlled inputs, reusable components, and React hooks like useState, useEffect, and useId.

### ✨ Features

- 🔄 Convert between any two currencies

- 🌐 Fetches live exchange rates using API

- 🔢 Precise input handling with controlled components

- ♻️ Reusable InputBox component

- 🎣 React hooks used:

- useState → state management

- useEffect → API calls + data sync

- useId → unique IDs for form inputs

### 🧠 What I Learned

- How to structure reusable React components

- How to manage multiple states (amount, fromCurrency, toCurrency, result)

- Working with API calls inside useEffect

- Ensuring unique form IDs using useId

- Controlled inputs & component re-render pattern
